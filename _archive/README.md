# Archive

This folder contains files and directories that are not actively used in the ARL Hockey landing page project.

## Contents

- **pnpm-lock.yaml** - Lock file for pnpm package manager (project uses npm)
- **examples/** - Third-party integration examples (Stripe, etc.)
- **supabase/** - Supabase edge functions and configuration (not currently used)

These files have been archived to keep the main repository clean and focused on the active codebase. They can be restored if needed in the future.
